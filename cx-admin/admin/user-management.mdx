### **2. 임시 계정 확인**

![](/Untitled\(2\).png)

![](/Untitled\(3\).png)

회원 가입 진행 중 이탈한 경우, 어드민에서는 계정이 검색되지 않으나 재가입을 시도하면 ‘이미 가입된 계정이 있어요’ 라는 팝업이 뜨며 가입이 불가할 수 있습니다.

이는 Firebase Console 에서는 계정이 생성된 걸로 인식했지만 실제로 완료되지 않아 ‘임시 계정’ 상태가 되었기 때문입니다.

**가입을 시도하신 이메일을 어드민의 \[임시 계정 확인] 탭에서 검색**하면 확인이 가능하며 임시 계정으로 확인된 경우, [임시 계정 삭제 요청](https://www.notion.so/16602c6645804536bf199807711afd0f?pvs=21) 에서 해결방법을 참고하시면 됩니다.

### 3. **탈퇴 사용자 복구**

![](/Untitled\(4\).png)

![](/Untitled\(5\).png)

Class101 가입 계정을 탈퇴한 경우 탈퇴 일로부터 **15일 이내** 계정 탈퇴를 철회할 수 있습니다. 고객센터로 탈퇴 철회 요청 문의가 인입된 경우, \*\*UID(=Firestore ID, Firebase ID)를 사용자 토큰에 기재 후 \[복구하기]\*\*를 통해 탈퇴 계정을 복구할 수 있습니다.

다만, 탈퇴 후에는 사용자 관리 탭에서 UID를 조회할 수 없습니다. `6. [탈퇴 예정자 관리](<https://www.notion.so/6026a34407a942e89cfed1ad798c25dc?pvs=21>)`에서 조회 후 복구를 진행해주세요.

### **4. 사용자 관리 유저 상세 페이지**

![](/Untitled\(6\).png)

필터를 추가 하면 조건에 맞는 유저의 정보를 확인할 수 있고 액션(✏️) 을 클릭하면 해당 유저의 상세 정보까지 조회가 가능합니다.

![](/Untitled\(7\).png)

* **윗 부분에 있는 기능들**

  1. **알림 전송 내역** : 유저한테 발송된 푸시, 문자, 메일 내용을 확인할 수 있습니다.

  2. **로그인 기록** : 등록된 기기, 최근 90일간의 접속 기록을 확인할 수 있습니다. (지역 정보는 정확성 떨어짐)

  3. **이메일 로그인 계정으로 변경** : 카카오 / 네이버 연동 계정 타입을 일반 로그인 방식으로 변경하는 기능입니다. (그 외의 계정은 변경이 불가하기 때문에 위 사진처럼 비활성화 되어 있음)

  4. ~~포인트 발급 : 미사용~~

  5. **수강권 이전** : 다른 계정으로 수강을 희망하는 경우 수강권 이전을 통해 결제한 클래스의 수강권을 이전할 수 있습니다. (구독을 이용중인 경우 불가)

  6. ~~친구초대 보상 지급 : 미사용~~

  7. **휴대폰 인증 해제** : 계정에 등록된 휴대폰 번호의 인증 해제 (새 계정 가입 시 기존 계정에서 인증 해제 해줘야함)

  8. **학생 인증 등록** : 학생 할인 프로모션 (KR만 사용)

  9. **비밀번호 재설정 링크 생성** : 클래스101 자체 계정의 비밀번호 변경 요청 시 전달 (구글/네이버/카카오톡 등 연동 계정 불가)

  10. **강제 로그아웃** : 로그인 되어있는 모든 기기에서 로그아웃 (유저 요청 시 사용)

  11. **회원 탈퇴**

### 5. 로그인 제한 관리

![](/Untitled\(8\).png)

* 기기 등록 대수가 4대를 초과하여 로그인이 제한된 유저는 해당 페이지에서 ‘제한 해제’ 할 수 있습니다.

* UID를 누르면 개인 상세 페이지로 넘어갑니다. 로그인이 제한된 사유, 90일 간의 접속 기록을 확인할 수 있습니다.

### 6. 탈퇴 예정자 관리

![](/Untitled\(9\).png)

회원 탈퇴 진행 시 15일 간 유예 기간이 주어지며, 유예 기간 동안에는 \[사용자 관리]탭에서는 정보가 조회되지 않기 때문에 \[탈퇴 예정자 관리] 탭에서 계정을 조회할 수 있습니다. 탈퇴 철회를 희망하는 유저의 경우 해당 페이지에서 \[탈퇴 철회]를 진행하면 됩니다.
